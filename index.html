Help Guide: How to Use Herotech

This guide will walk you through getting started with Herotech and explain the functionality of each navigation option. Herotech serves as a central hub linking to specialized tools. To use the platform:

  • Getting Started: Click on any card above to access the respective tool. External tools open in a new tab. No account is required for basic access, but some features may prompt registration.
  • Browser Compatibility: Use a modern browser like Chrome, Firefox, or Edge for the best experience.
  • Troubleshooting: If a tool doesn't load, check your internet or refresh. Refer to individual tool docs for support.
  • Dark Mode: Toggle the mode in the header for your preferred viewing experience.
  • Mobile App: Download the Herotech app from the button in the header for on-the-go access.
